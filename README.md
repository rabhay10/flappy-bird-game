# Simple_Flappy_Bird_Game
Flappy Bird Clone Walkthrough I have created a complete, standalone Flappy Bird clone in a single HTML file.

Features
60 FPS Physics: Smooth gravity and flap mechanics using requestAnimationFrame.
Procedural Graphics: All assets (bird, pipes, clouds, ground) are drawn using Canvas API. No external images.
Audio: Synthesized sound effects using Web Audio API (no external files).
Mobile Friendly: Touch controls, responsive resizing, and prevented default touch actions.
Polish: Particle effects for trails and explosions, parallax scrolling background, and high score persistence.

Gameplay: The bird falls with gravity and jumps with impulse.

Collision: Hitting pipes or the ground triggers Game Over.

Scoring: Passing pipes increments score. High score is saved.

Performance: The game loop is optimized to run at 60 FPS.

How to Run:

1.Open the file >> "index.html"
  in any modern web browser.
  
2.Click, tap, or press Space to play.
